1. Open GitHub and go to the page for your prototype

2. Click pull requests and open a new pull request

3. Select your branch as the one to be compared with the main branch

4. Choose a code reviewer

5. Merge the pull request

If the code reviewer finds no issues, merge your branch with the main branch.

6. How to pull the latest merged changes if you're working on a new branch

In your terminal make sure you're on your new branch and run:

git merge main
