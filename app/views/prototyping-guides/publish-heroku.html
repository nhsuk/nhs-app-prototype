1. Make sure you're in the right folder

If you named your clone remote-lab for example, use the terminal to navigate to that folder. To do this, run:

cd remote-lab

2. Login to Heroku

Run this command:

heroku login

3. Choose a name and create your Heroku app

This is the name that will be used in the URL of your Heroku app. For example, prescriptions-lab.

For this example, you would run this command:

heroku apps:create prescriptions-lab --region eu

4. Push to Heroku

Make sure you commit any changes to git before deploying. But do not push your changes.

Run this command:

git push heroku main

5. Open Heroku

Run this command:

heroku open
