1. Make sure you're in the right folder

Go to the folder where you would save your prototypes, for example, your home folder. To do this, run:

cd folder name

2. Clone the main repo

Run this command:

git clone https://github.com/nhsuk/nhs-app-prototype.git nhs-app

This will clone the main repo in a new folder called nhs-app.

3. Navigate to your new folder

Run this command:

cd nhs-app

4. Install the kit

Run this command:

npm install

This might take a few minutes.

5. Run the kit

Run this command:

npm run watch

6. Open the kit locally

Check the kit is working by running it locally on your own machine.

Open local host
