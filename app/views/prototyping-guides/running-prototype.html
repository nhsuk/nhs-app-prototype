1. Make sure you're in the right folder

If your prototype was named practice-kit for example, use the terminal to navigate to that folder. To do this, run:

cd practice-kit

2. Run the kit

Run this command:

npm run watch

3. Open the local host link in your terminal

Your prototype should now be running locally.

Your terminal should show a 'localhost' link. For example:

http://localhost:3000

Open that link in your browser.
